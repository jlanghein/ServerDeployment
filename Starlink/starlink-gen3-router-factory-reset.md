# Starlink Gen 3 Router Factory Reset Guide

This guide covers how to factory reset a Starlink Gen 3 (Standard) router to restore it from Bypass Mode back to default settings.

---

## When to Reset

- Router is in Bypass Mode and you want to restore WiFi functionality
- Troubleshooting connectivity issues
- Preparing the router for a new location or owner
- Restoring default settings after configuration changes

---

## Method 1: Factory Reset via Starlink App (Preferred)

### Prerequisites
- Starlink app installed on your phone
- Router powered on and connected to the dish

### Steps

1. **Connect to Starlink network**
   - If Bypass Mode is enabled, you'll need to connect a device directly to the router via Ethernet
   - Or temporarily disable Bypass Mode first (see Method 2)

2. **Open the Starlink app**

3. **Navigate to Settings**
   - Tap the gear icon or **Settings**

4. **Find Factory Reset option**
   - Go to **Network** > **Factory Reset**
   - Or look under **Advanced** > **Factory Reset**

5. **Confirm the reset**
   - Tap **Factory Reset** and confirm when prompted

6. **Wait for the router to reboot**
   - The router will restart automatically
   - This may take 2-5 minutes
   - WiFi will be restored with default SSID and password

---

## Method 2: Physical Factory Reset (Hardware Button)

Use this method if you cannot access the Starlink app or the router is unresponsive.

### Locate the Reset Button

The Gen 3 router has a reset button on the bottom of the unit.

### Steps

1. **Ensure the router is powered on**
   - The router should be connected to the dish and receiving power

2. **Locate the reset button**
   - Look on the bottom of the router for a small pinhole or recessed button

3. **Press and hold the reset button**
   - Use a paperclip or SIM ejector tool
   - Hold for **15-20 seconds**

4. **Wait for LED indicators**
   - The router lights will flash or change pattern
   - Release the button when you see the lights cycle

5. **Wait for the router to reboot**
   - The reset process takes 2-5 minutes
   - The router will broadcast the default WiFi network when ready

---

## Method 3: Power Cycle Reset (Soft Reset)

This does NOT factory reset but can help if the router is stuck.

1. **Unplug the power cable** from the router
2. **Wait 30 seconds**
3. **Plug the power cable back in**
4. **Wait for the router to fully boot** (2-3 minutes)

> Note: This only restarts the router; it does not change settings.

---

## After Factory Reset

### 1. Reconnect to Default WiFi

- **Default SSID**: `STARLINK` or `STINKY` (varies by unit)
- **Default Password**: Printed on the bottom of the router or in the original packaging

### 2. Open the Starlink App

- The app will detect the reset router
- Follow the setup wizard to configure:
  - WiFi network name (SSID)
  - WiFi password
  - Other preferences

### 3. Verify Bypass Mode is Disabled

- Go to **Settings** > **Network**
- Confirm **Bypass Mode** is set to **OFF**
- WiFi should now be active

### 4. Test Connectivity

- Connect a device to the new WiFi network
- Verify internet access

---

## Troubleshooting

### Router won't reset
- Ensure the router is receiving power from the dish
- Try holding the reset button longer (up to 30 seconds)
- Check that you're pressing the correct button (not the release latch)

### Can't find default WiFi after reset
- Wait 5 minutes for the router to fully initialize
- Move closer to the router
- Check if the dish has a clear view of the sky (router may not broadcast until connected)

### App doesn't detect the router
- Ensure your phone is connected to the Starlink WiFi
- Force close and reopen the app
- Check for app updates

---

## Notes

- Factory reset removes all custom settings including:
  - Custom WiFi name and password
  - Bypass Mode configuration
  - Any saved network preferences
- Your Starlink account and subscription are NOT affected
- The dish does not need to be reset; only the router settings are cleared
